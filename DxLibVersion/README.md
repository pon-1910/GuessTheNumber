# 数当てゲーム (DxLib版)

このプロジェクトは、C++のゲーム制作ライブラリである **DxLib** を使用して作成した、GUI版の数当てゲームです。
コンソール版で作成したロジックを継承しつつ、リアルタイムな描画とキーボード入力、視覚的な演出を追加しています。

## 🎮 ゲーム概要
- 1〜100（デフォルト）の中からランダムに選ばれた正解の数字を当てるゲームです。
- 入力した数字に対して「もっと大きい」「もっと小さい」といったヒントが表示されます。
- 正解した際には、点滅エフェクトによるお祝い演出が表示されます。

## ✨ 主な実装機能
- **リアルタイムループ**: `while` ループによる毎秒60フレームの描画管理。
- **動的なテキスト描画**: `DrawFormatString` を用いた変数（試行回数など）のリアルタイム表示。
- **中央揃え計算**: `GetDrawStringWidth` を利用した、画面幅に合わせたテキストの自動センタリング。
- **数値入力**: `KeyInputNumber` による、GUI上でのスムーズな数字入力。
- **リセット機能**: クリア後に `R` キーを押すことで、クラスインスタンスを再生成し、ゲームを即座にリスタート。

## 🛠️ 開発環境
- **言語**: C++
- **ライブラリ**: [DxLib (ＤＸライブラリ)](https://dxlib.xsrv.jp/)
- **IDE**: Visual Studio 2022

## 🚀 実行方法
1. `DxLibVersion.sln` を Visual Studio で開きます。
2. DxLib の環境構築（インクルードディレクトリ等の設定）が済んでいることを確認してください。
3. `F5` キー（またはデバッグ開始）で実行します。
