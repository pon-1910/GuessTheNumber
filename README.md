# GuessTheNumber (数当てゲーム)

C++の基礎学習の一環として作成した、シンプルな数当てゲームです。
書籍『やさしいC++』を読了後、最初の実践プロジェクトとして開発しました。

## 概要
コンピュータがランダムに選んだ0〜100の数字を当てるゲームです。
正解よりも大きいか小さいかのフィードバックを受けながら、正解を目指します。

## 現在の機能
- **乱数生成**: 実行ごとに異なる正解（0〜100）を生成します。
- **バリデーション**: 0〜100の範囲外の数字が入力された際のエラーチェック機能を搭載しています。
- **動的フィードバック**: 入力した数字が正解より大きいか小さいかをリアルタイムで表示します。

## 開発環境
- 言語: C++
- 開発ツール: Visual Studio 2022 
- コンパイラ: MSVC

## 実行方法
1. リポジトリをクローンまたはダウンロードします。
2. `main.cpp` をコンパイルします。
3. 実行ファイルを実行し、コンソール上の指示に従って数字を入力してください。
